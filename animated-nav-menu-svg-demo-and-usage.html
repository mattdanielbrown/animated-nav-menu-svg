Basic Usage

First, reference the stylesheet

Either as CSS in the HTML document:

<!-- Inside your <head> element --> 
<link rel="stylesheet" href="~/node_modules/animated-hamburger-menu-icon/animated-hamburger-menu-icon.css">

Or in your SCSS partial:

@import "~/node_modules/animated-hamburger-menu-icon/animated-hamburger-menu-icon.css";

Next, reference the Javascript file

Ususally at the end of your HTML document:

<!-- Wherever import your other javascript files... -->
<script src="~/node_modules/animated-hamburger-menu-icon/animated-hamburger-menu-icon.js"></script>

Lastly, add the element wherever your want it, usually in <nav>

Markup for use as a button element:

<!-- Navigation Menu Button (Toggable-SVG inside a button element) -->
<button type="button" id="menu-button">
  <svg xmlns="http://www.w3.org/2000/svg" viewBox="0 0 32 32" width="32" height="32" id="menu-icon">
    <title>animated-hamburger-menu-icon-svg</title>
    <rect y="4" width="32" height="4"></rect>
    <rect y="16" width="32" height="4"></rect>
    <rect y="28" width="32" height="4"></rect>
  </svg>
</button>
